# Wu et al., 2021

Authors: Mingzheng Wu, Samuel Minkowicz, Vasin Dumrongprechachan, Pauline Hamilton, Lei Xiao, Yevgenia Kozorovitskiy*

## Dependencies
The Python code depends on numpy, scipy, pandas, matplotlib, and graphviz.
